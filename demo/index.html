Credentials and destination

Mount form 1

Programmatic mount()

Calls SignalWireAddressWidget.mount('#target-1', {token, destination}). The widget mounts into the target div. The div's existing children (here, none) would be wrapped inside the launcher.

Mount form 2

Declarative <signalwire-address>

The custom element is auto-registered on import. Attributes map to the same options. This instance passes its own slotted content (the span below) instead of using the default label.

Mount form 3

Audio-only mode

Passes video: false. Mic permission only, camera button hidden, and the 16:9 video area collapses entirely. Pass poster to show a custom image in its place.

Mount form 4

Stacked layout

Passes layout: 'stacked'. Video sits smaller at the top (capped on desktop) with the transcript flowing beneath. The self-preview overlay (default on) shows in the bottom-right of the video frame — set showLocalVideo: false to hide it.
